出去看电影

学车

发表于 2020-08-10 |

懒癌又犯了，又有几天没更了。
写一下学车的经历吧，正好也可以给后面的同学一些参考。
我还没学完，先写到我的进度，学完再更新一下。
2020/8/10：科目二已过，等待科目三

驾校

人在北京，去的东方时尚（只是为了和同学一起，多点伴）。

报名

东方时尚有好多报名点，拿地图一搜就有了。带着身份证和白底1寸照片，直接去报名点，接下来按着工作人员的指导就行了。他应该会让你下两个App：交管12123（约考试的），东方时尚（看网课，做模拟题，扫码签到签退）。

我是报的学生速成，3个人拼单可以便宜一点：4800元。

科目一

报完名之后就该看网课了，看完网课才能去12123App约科目一考试，约也只能约好几天之后的。想节约时间的同学可以赶紧刷完网课，趁早报名，约考和真考之间的那几天再慢慢地学（那几天的时间很足够了）。先把题库里的1500道刷一遍，再做做模拟测试，基本上就稳了。

约考后要注意短信和东方时尚App的通知，约考有可能不成功（被已经没约成一次的顶走了），报的时候注意你的排名，再约一次还不如约之后的省时间。约考成功后会有短信通知，考试前东方时尚App内的通知会告诉你具体的事项。

等到考试那天，别忘了身份证，早点去驾校（坐班车到达的时间就行，不用特意提早去），下了车赶紧体检（如果报名时已经体检了就不用了，大多都得体检），不然待会队就排起来了。体检得准备20元钱，就草草地测个视力，录个信息，之后可以去考场了。如果饿了也可以先吃饭，但要注意时间。

进了考场那座楼，上二楼。之后的流程如下：
1.扫健康宝，测体温，领一次性手套（手套没什么用，电脑使完就消毒了）。
2.刷身份证，从机子上取号。
3.排队录入指纹。
4.找座就座，听考试须知。
5.等叫号，参加考试。
6.签字完成考试。
<以下是合格之后的事了>
7.一楼约车（预约科目二的训练）
8.庆祝阶段性的胜利😝

提醒：
考试前关好手机，尤其注意是否有闹铃等其他可能导致中途开机的东东。
考试的题目字体有点小，一定要看清楚。
考试是一做一判的，如果错了会有窗口弹出，不要慌，一定要稳住心态。

科目二

训练的前两次是机器人或VR二选一，之后才是真人教练。

我前两节课选得VR，因为没玩过，正好体验一下。就是画质一般，一点也不逼真（能学会东西就行了）。那两节课没有太多东西，主要是基础，练打轮，练半联动。注意提前找好教室，两次不一定是一个地方，别走错了。

后面的真人课是两个学员一个车。（我赶得很巧，我同车的小伙伴和我住得很近，更巧的是我们俩生日只差一天。这就是缘分吧😂）要商量好应该也可以两人一起报一个车。随机的话，一般是同性，假如真是碰到了异性，全当是驾校给你牵线了🤪。真人课时记得先去签到处（大多都在太阳伞底下，找太阳伞就行），扫健康宝，登记体温，领个体温正常的小卡片。（这是疫情期间的措施，疫情过后就不知会什么样了）

约完车后会给你一张小票，告诉你什么时候约科目二考试，别忘了就行。约完考试，和科目一一样要注意短信和App。考试前有个测试，让你有机会跑跑考场，用一下考试车。一般在考试前一天。

科目二有五个项目：侧方，坡起，直角，曲线，倒库
基本就是背点位，控好速度就行。只有曲线没什么点位，靠感觉来就行。

科目二的考场挺明显的，从车站往北走就能看见。就不放图了（要想看图，可以参考上面的图，里面的”备考大厅“就是考场）考完试，签个字，就能回家打电话约科目三的训练了。

科目三

进入了我的知识盲区。。。。

VBS

Vbs是一种Windows脚本，它的全称是:Microsoft Visual Basic Script Editon.(微软公司可视化BASIC脚本版)，VBS是Visual Basic的的一个抽象子集，是系统内置的，用它编写的脚本代码不能编译成二进制文件，直接由Windows系统执行（实际是一个叫做宿主host的解释源代码并执行），高效、易学，但是大部分高级语言能干的事情，它基本上都具备，它可以使各种各样的任务自动化，可以使你从重复琐碎的工作中解脱出来，极大的提高工作效率。
