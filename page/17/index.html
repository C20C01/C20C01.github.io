新年快乐！！！

赛博朋克2077

发表于 2020-12-19 | | 阅读次数:

又过了两周，高中的同学已经有好多放假回家的了。我们还得坚守到明年1.8。

12.10前几天就开始预载了，然而因为流量不够就只能等回家了。也没有第一时间玩上，但也就晚了1天。

2080S能跑中等光追，散热风扇一直最大转速。好吃配置。

Bug好多，到不太影响正常游戏（只是不会卡关，但是搞你心态）第一回开了一个档，玩了几个小时。做任务时被搞死了，返回上一检查点时游戏直接崩溃，存档直接没掉。（期间我没手动存档，不确定手动存档能不能挨住这一下）总之还是多多存档吧（脱离卡死，体验剧情…）。我其实挺喜欢良性bug的，很快乐😂

打人的话有一个阴招，充能技术步枪，大狙都是可以穿墙击杀的（应该是随便穿，没有打不到的时候），配上一个能显示敌人位置的义眼或者瞄准镜也可以黑个摄像头。找个好位置就能爽起来了，不要离敌人太近，否则他可能借着强化剂跑贼快，直接骑你脸。反正你隔着多少墙都能干。

搞钱的话可以卖油画，4000钱卖出去，5钱买回来。能一直刷，详情移步B站。

不知道上面那俩算不算bug，会不会修复，反正这俩爽到我了😜

19岁生日

发表于 2020-12-05 | | 阅读次数:

2001-12-04 ~ 202012-04 🎈CC19岁了🎈

今年生日的日子赶的不错，正好是个周五。可以回家爽一爽。

放了学，和高中同学一起回去看老师。到家时天都黑了，收拾了收拾就去外面下馆子。吃的“小手套”（我对鱼肚的爱称）再例行一碗面，要生日面直接就送了一碗，味道都还不错。姥姥还给了个红包。除此之外就没什么和平日不一样的了。

第二天，宅了一天家。管爸妈要生日礼物，貌似一直以来没要成过（要送的时候都是直接给，都不用要。也可能因为礼物一直想要的是玩具，嫌我太老，不给买了😂）今年很实在，直接发红包了。想着1月7号放的寒假，想买点之后解闷的东东。

哈哈哈，正好赛博朋克2077下周就出了（只要不接着跳就是了）为游戏事业第一次做出了实质性的贡献（在游戏方面花钱，我可是老白嫖、老零充了）开了个好头啊，希望能让我爽一爽。（话说支付真的好方便，还以为要用银行卡呢）

❤最后感谢各位19年以来的陪伴❤
