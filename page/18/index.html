我的新设备

发表于 2020-08-06 |

又有几天没上新了，可不是我懒😏
这几天一直都在摆弄新玩意。

旧手机的扑街：

我的旧手机（OPPO R9S Plus）突然暴毙。学完车刚要扫码签退，就在点扫一扫的按钮时候，手机先是卡住不动，然后直接黑屏开始无限重启。一直持续了20min左右，终于开开了。虽然也使了三年了，但还没有出过什么问题。所以第一反应不是手机坏了，感觉可能是中了什么毒，于是上来就开始杀毒，结果又黑了。这一黑，就又打不开了……除了手机其他的东西也没带，只能走回家。好在驾校班车的站离家不算太远，但是顶着大太阳，热的一批。搞得这个人都不嘚了，不争气的机还在震动着（手机开机的震动，遇到无限重启😥）疯狂地搞我心态。

新手机&新表的选购：

有了电脑之后，就没怎么玩手机了。所以开始是本着不打游戏，买个能花钱，能拍照，能聊天的千元机。看着觉得荣耀就可以，不算太贵，配置还可以。但到了店里都没有那款，最后选了华为的nova7，因为一直没试过曲面屏，再说配置好像没啥差别，所以没买pro。后来又买了个表，以后看点、刷卡啥的方便点。

手机支持5G，麒麟985芯片，OLED显示屏，没耳机孔（不嘚😥），四个后置摄像头（我也不知道都是干啥的），40W快充。使着满嘚的。

表上可以建一个公交卡，直接刷。说是可以息屏刷卡，应该是NFC功劳。连上手机的蓝牙就能同步消息了，收个微信什么的，就会有提醒，还能看见发的什么。但是功能比较少，无法下软件。还接不了微信电话（只能接电话…）。几天下来，感觉还不赖，可穿戴智能设备还是很方便的。

我的新电脑

发表于 2020-08-03 |

HP家的 暗夜精灵6 plus

说是新电脑，不知不觉间也使了小一个月了。

在2020-07-13 00:00:23从京东上的自营店下的单，至于为什么要在大半夜的下单，那一定是有活动啊。零点的时候HP出新款，所以有一波活动。我这个虽不是新款，但也有优惠。其实优惠并不是深夜下单的主因，真正的主因是没货。我那款除了活动基本上都是缺货的状态，只能趁着活动有货抢了😂

配置：

  • 处理器：Intel(R) Core(TM) i9-10885H CPU @ 2.40GHz
  • 显卡： GeForce RTX 2080 Super（8G）
  • 内存：16Gx2
  • 储存：1T固态 2T机械
  • 屏幕：17.3英寸300Hz刷新率

价格：

原价20999，活动减掉1000，再加上10元优惠券，到手价19989
这可是我第一个上万的电子产品。

大小：

本灵魂画手要发功了

性能：

性能很棒，可以全特效流畅运行绝大多数游戏。

  • 我的世界 特效全开（然而没装光影水反） 平均帧数在144左右

  • 地平线4 使用超高的预设 平均帧数在170左右

  • 看门狗1 特效全开 平均帧数可达300

  • Apex Legends 特效全开 平均帧数可达300

开机只需要5s左右，浏览器冷启动只需要1s左右，比以前的老爷机快得不知哪去了，固态就是爽！

散热：

散热也很棒，应该是静音风扇，相同进气量声音应该小很多。玩游戏时也就70~80℃ 而且也只是局部小热，并没有那种“铁板烧”的感觉。本子下面有很大的进气口（占了底盘的小一半）。总之不用担心散热方面的问题（反正目前我没有遇到）

便携性：

便携性应该不用说了吧，巨大的适配器，330W的，也不算轻。一只胳膊端不动的本体（习武之人专用），反正像办公本一样随时随地掏出来使你就别想了。但是买这种本的，便携性肯定是次要的啦🤪

其他：

键盘灯光有4个区域，一个区域内是一个颜色。键盘上还有6个可编程的按键，我没怎么用过，不知好不好使。

和大多游戏学习本一样，这个也自带一个控制中心——HP OMEN Command Center，感兴趣的可去自行了解，我就不做过多解释。

爽就完事了😁
