import java.util.Random;
public class TwoColorBall {
    public static void main(String[] args) {

        Random ee=new Random();//篮球随机
        int[] red=new int[33];//生成数组

        for(int i=0;i<33;i++) {
            red[i]=i+1;
        }//填充数组

        System.out.print("RedBalls: ");

        for(int i=0;i<6;i++) {
            a=dd.nextInt(b-i);
            System.out.print(red[a]);
            red[a]=red[32-i];
            //1号~33号抽一个
            //将被抽到的数覆盖为33
            //1号~32号抽一个 
            //将被抽到的数覆盖为32
            //1号~31号抽一个 
            //......

            if (i<5) {
                System.out.print("   ");
            }
        }//打印红球

        System.out.println();
        System.out.print("BlueBall: ");
        c=ee.nextInt(16)+1;
        System.out.print(c);//打印蓝球

}

在禁用粘贴的网页上进行粘贴

发表于 2020-10-24 |

java上机课的时候,得把代码搞进学习通布置的作业里.然而那天老师忘了勾选”允许复制”,于是没法直接把IDE里的代码直接复制到作业里了.虽说是熟练一下代码,但我这么懒的人,怎么可能再敲一遍.于是想到了浏览器的F12(直接修改网页,可以实现很多骚操作)

先在输入栏里打几个字(方便寻找文本框背后代码的位置),再F12选中刚才瞎打的字,直接双击进行修改,Ctrl+C Ctrl+V就可以起飞了.

其实这个方法也可以用于禁止复制的网页,毕竟网页文件都到你手里了😜
